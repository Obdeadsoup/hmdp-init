# Agent 提示词模板

这些模板可以直接复制给 ChatGPT / Codex。复制后把阶段、文件、报错替换成你的真实内容。

## 1. 开始一个阶段

```text
请读取 AGENT.md 和 docs/learning/当前阶段文档。
我现在进入阶段 X：模块名称。

请先不要改代码，先完成：
1. 判断当前任务属于哪个学习阶段。
2. 阅读相关文件并给文件地图。
3. 用初学者能懂的话解释核心概念。
4. 给出 3 到 5 步最小实现计划。
5. 告诉我每一步如何验证。
```

## 2. 实现一个小功能

```text
当前项目是黑马点评后端学习项目。
当前阶段：阶段 X。
本次目标：只实现 XXX。

任务边界：
- 本次只改这些文件：...
- 不做：...

要求：
1. 先阅读相关文件。
2. 给出文件职责说明。
3. 给出最小修改计划。
4. 再修改代码。
5. 修改后运行 mvn -q -DskipTests compile。
6. 给出 curl 或 .http 验证方式。
7. 总结我必须掌握的知识点。
```

## 3. 让 Agent 审查代码

```text
请按代码审查模式检查我当前实现。

背景：
我实现了阶段 X 的 XXX 功能。

重点检查：
1. 功能是否正确。
2. 边界条件是否完整。
3. 是否符合 Controller-Service-Mapper 分层。
4. 是否有事务问题。
5. 是否有并发问题。
6. 是否有缓存一致性问题。
7. 是否泄露敏感信息。
8. 是否需要补测试。

请按“必改问题、建议优化、背后的知识点、验证建议”输出。
```

## 4. 排错

```text
我在黑马点评项目遇到错误，请按 AGENT.md 的排错流程帮我定位。

当前阶段：
...

我刚改过的文件：
...

执行的命令：
...

接口请求：
...

关键报错：
...

请输出：
1. 错误类型。
2. 关键异常行。
3. 错误含义。
4. 最可能原因，按概率排序。
5. 逐步验证方式。
6. 最小修复。
7. 如何以后避免。
```

## 5. 让 Agent 出题

```text
我刚学完阶段 X：XXX。
请你根据本阶段内容考我：
1. 5 个概念题。
2. 2 个代码阅读题。
3. 1 个小改造任务。
4. 1 个面试表达题。

先只给题目，不要给答案。等我回答后再批改。
```

## 6. 让 Agent 生成接口测试记录

```text
请根据当前项目接口，帮我补充 docs/api/hmdp.http。

要求：
1. 按阶段分组。
2. 每个接口包含 Method、URL、Header、Body。
3. 需要登录的接口使用 @token 变量。
4. 给出预期响应说明。
5. 不要写真实 token 或密码。
```

## 7. 让 Agent 生成面试表达

```text
请基于我当前实现的黑马点评项目，帮我整理面试表达。

要求：
1. 30 秒版本。
2. 1 分钟版本。
3. 模块亮点。
4. Redis 数据结构选择。
5. 缓存问题和解决方案。
6. 秒杀问题和解决方案。
7. 面试官可能追问和参考回答。
```

## 8. 让 Agent 做学习复盘

```text
请帮我复盘阶段 X：XXX。

输出：
1. 本阶段目标。
2. 我改过的文件。
3. 核心调用链。
4. 关键代码解释。
5. 常见坑。
6. 验证方式。
7. 面试表达。
8. 下一阶段准备。
```
