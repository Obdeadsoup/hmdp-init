# 阶段 8 练习题：工程化与面试表达

## 概念题

1. 单元测试、集成测试、接口测试有什么区别？
2. 为什么配置要分环境？
3. Docker Compose 在学习项目中解决什么问题？
4. README 应该写哪些内容，才能帮助别人运行和理解项目？
5. 简历项目描述为什么不能只写技术栈？

## 代码阅读题

1. 阅读 `pom.xml`，找出测试依赖和 Spring Boot 打包插件。
2. 阅读 `application.yaml`，说明哪些配置应该改成环境变量。

## 小改造任务

整理一个完整的接口测试记录，至少包含登录、商铺详情、点赞、签到、秒杀。

## 面试表达题

用 1 分钟介绍整个黑马点评项目，要求包含业务、技术栈、亮点和你解决的问题。
